Propósito de la Pagina

Esta pagina tiene el proposito de mostrar reseñas de diferentes materiales de diversas categorias ya sean Videojuegos, Peliculas y Series. Esto con la finalidad de ayudarle al usuario a que pueda tomar una mejor decision con base a una reseña para saber en que va a utilizar su tiempo por supuesto que estas reseñas no son recomendaciones de que ver o no hacer o que jugar o no jugar esto es meramente para que el usuario se atraiga con la ayuda de una reseña rica en expresividad,etc.

¿Que deberia esperar de esta Pagina web?

Esta pagina esta solo con el objetivo de informar por lo que lo unico que se puede llegar a esperar de ella es que informe correctamente por medio de las reseñas que se colocan en las imagenes de los diversos contenidos de las diversas categorias que se manejan en la pagina web

Para que se uso Bootstrap 5 en esta pagina web

Se uso Bootstrap 5 en esta pagina web para que sea responsive con ayuda de los container, tambien se uso bootstrap 5 para agregarle y quitarle diseño a los enlaces, ya que con bootstrap por defecto los enlaces tienen una forma predefinida
